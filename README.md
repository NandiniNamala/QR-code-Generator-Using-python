# QR Code Generator Using Python
# Project Overview

This project is a QR Code Generator built using Python and Tkinter. It allows users to generate QR codes from both text and URLs. The application provides a simple graphical interface with QR preview and the ability to save the generated QR code as a PNG image.

## Features

- Generate QR codes from URLs
- Generate QR codes from text
- QR code preview
- Save QR code as PNG
- Copy Text/URL button
- Clear input button
- URL validation
- User-friendly GUI

##  Technologies Used

- Python
- Tkinter
- qrcode Library
- Pillow (PIL)

##  Project Files

- `gui_qr_generator.py`
- `logo.png`

## Installation

Install the required libraries:

```bash
pip install qrcode pillow
```
##  Run the Project

```bash
python gui_qr_generator.py
```
## Future Enhancements

- Custom QR colors
- Dark Mode
- QR Code history
- Logo inside QR Code
- Multiple QR size options

## output 
The application Generates a QR code and saves it as a PNG image that can be Scanned using any QR code Scanner.
